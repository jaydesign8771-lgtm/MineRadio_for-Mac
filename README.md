# MineRadio for Mac

MineRadio for Mac 是基于开源 Windows 版 Mineradio 二次开发的 macOS 桌面音乐播放器。这个分支保留原项目的沉浸式视觉、歌词舞台、3D 歌单架和在线音乐体验，并围绕 Mac 本地使用习惯做了适配。

当前版本：`1.1.1`

## 项目说明

本仓库是 Mac 改造版工程，重点是让 Mineradio 在 macOS 上更像一个原生桌面音乐 App：可以导入本地音乐文件夹，识别同目录歌词文件，使用 macOS 风格窗口控制，并提供 DMG 打包配置。

## 主要改动

- 增加 macOS 打包配置，支持生成 `Mineradio-1.1.1.dmg`。
- 增加 macOS 风格窗口控制按钮和窗口图标适配。
- 增加系统托盘/菜单栏基础操作入口。
- 首页入口调整为「本地导入」，左侧歌单区增加「本地导入」入口。
- 导入本地文件夹时支持递归扫描多级目录，并按子文件夹组织本地歌曲。
- 自动识别本地歌词文件，支持 `.lrc`、`.txt`、`.text`、`.lyric`、`.lyrics`。
- 播放页增加歌词快捷控制条：原文/译文/双语、歌词时间偏移、歌词大小调整。
- 优化双语歌词显示距离，让译文更贴近原歌词。
- 左侧歌单列表去掉「当前队列」入口，减少播放后自动跳转。
- 左侧歌曲列表优化当前播放状态，增加动态频谱提示，并弱化条目边框。
- 首页「施工」模块改为听歌报告/音乐基因方向，展示听歌时长、偏好和心情指数等数据入口。

## 核心特性

- 沉浸式播放页，包含歌词舞台、粒子视觉和封面视觉联动。
- 基于节奏分析的电影镜头视觉系统。
- 3D 歌单架，用于浏览歌单和播放队列。
- 本地音乐文件夹导入、本地歌词识别和本地封面展示。
- 歌词翻译显示模式、歌词时间偏移和字号快捷调节。
- 网易云音乐账号、搜索、歌单、播客等体验接入。
- QQ 音乐搜索、登录态与音源补充接入。
- Open-Meteo 天气电台，根据天气和城市生成播放队列。
- 自定义专辑封面、自定义歌词、视觉参数和用户存档。
- GitHub Releases 更新检测入口。

## 安装与运行

开发运行：

```bash
npm install
npm start
```

构建 macOS DMG：

```bash
npm run build:mac
```

构建产物位于 `dist/`。正式分发建议使用 DMG，不建议直接分发 `dist/mac-arm64` 目录。

## 安全与隐私

登录 Cookie、搜索历史、本地导入路径、自定义封面、自定义歌词和节奏分析缓存应只保存在用户本机，不应提交到仓库。

本仓库已忽略常见本地数据：

- `.cookie`
- `.qq-cookie`
- `.env`
- `dist/`
- `node_modules/`
- 本地备份和节拍缓存目录

更多说明见 [PRIVACY.md](./PRIVACY.md) 和 [SECURITY.md](./SECURITY.md)。
